# Image Assets Needed

The following images need to be added to complete the Figma design implementation:

## Hero Section
- `/public/michelangelo_hand.png` - The Michelangelo "Creation of Adam" cropped hand image
  - Size: ~228px x 49px (cropped oval)
  - Can be extracted from the Figma design

## Skills Section
The following images should be placed in `/public/services/`:

1. `/public/services/ux_research.png` - UX Research illustration with decorative frame
2. `/public/services/wireframes.png` - Wireframes illustration with decorative frame  
3. `/public/services/prototyping.png` - Prototyping illustration with decorative frame
4. `/public/services/usability_testing.png` - Usability Testing illustration with decorative frame

Each image should be approximately 200px x 242px with the decorative frame included.

## How to get these assets:

You can extract these from the Figma file using the Figma Desktop MCP tools:
- Use the node IDs from the design to export specific elements
- The assets are available via the localhost server when using get_design_context

Alternatively, you can:
1. Open the Figma file
2. Select each asset layer
3. Export as PNG at 2x resolution
4. Place in the appropriate public folder
